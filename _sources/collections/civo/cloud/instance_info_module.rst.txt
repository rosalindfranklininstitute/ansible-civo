.. Title

civo.cloud.instance_info module -- Gather information about Civo compute instances
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://github.com/rosalindfranklininstitute/ansible-civo>`_ (version 0.0.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install git+https://github.com/rosalindfranklininstitute/ansible\-civo.git`.

    To use it in a playbook, specify: :code:`civo.cloud.instance_info`.

.. rst-class:: ansible-version-added

New in civo.cloud 0.0.1

Synopsis
--------

.. Description

- Returns details of one or all Civo compute instances in a region.
- When :emphasis:`hostname` is given, returns a single\-item list (or empty list if not found).
- Uses the :literal:`civo` CLI binary on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Get all instances
      civo.cloud.instance_info:
        region: LON1
      register: vms

    - name: Get a specific instance
      civo.cloud.instance_info:
        region: LON1
        hostname: web-01.example.com
      register: vm

    - name: Print public IP
      ansible.builtin.debug:
        msg: "{{ vm.instances[0].public_ip }}"
