.. Title

civo.cloud.snapshot module -- Manage Civo instance snapshots
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://github.com/rosalindfranklininstitute/ansible-civo>`_ (version 0.0.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install git+https://github.com/rosalindfranklininstitute/ansible\-civo.git`.

    To use it in a playbook, specify: :code:`civo.cloud.snapshot`.

Synopsis
--------

.. Description

- Create, delete, or restore point\-in\-time snapshots of Civo compute instances.
- Uses the :literal:`civo instance snapshot` CLI commands on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Create a snapshot of an instance
      civo.cloud.snapshot:
        name: web-01-snap
        instance: web-01
        description: "Pre-upgrade snapshot"
        region: LON1
        state: present
      register: snap

    - name: Restore snapshot to a new instance
      civo.cloud.snapshot:
        name: web-01-snap
        instance: web-01
        hostname: web-01-restored
        region: LON1
        state: restored

    - name: Restore snapshot overwriting the original instance
      civo.cloud.snapshot:
        name: web-01-snap
        instance: web-01
        overwrite_existing: true
        region: LON1
        state: restored

    - name: Delete a snapshot
      civo.cloud.snapshot:
        name: web-01-snap
        instance: web-01
        region: LON1
        state: absent
