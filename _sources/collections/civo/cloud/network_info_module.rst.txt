.. Title

civo.cloud.network_info module -- Gather information about Civo private networks
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://github.com/rosalindfranklininstitute/ansible-civo>`_ (version 0.0.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install git+https://github.com/rosalindfranklininstitute/ansible\-civo.git`.

    To use it in a playbook, specify: :code:`civo.cloud.network_info`.

Synopsis
--------

.. Description

- Returns details of one or all Civo private networks in a region.
- When :emphasis:`name` is given, returns a single\-item list (or empty list if not found).
- Uses the :literal:`civo` CLI binary on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Get all networks
      civo.cloud.network_info:
        region: LON1
      register: nets

    - name: Print network names
      ansible.builtin.debug:
        msg: "{{ nets.networks | map(attribute='label') | list }}"

    - name: Get a specific network
      civo.cloud.network_info:
        region: LON1
        name: my-network
      register: net

    - name: Print network ID
      ansible.builtin.debug:
        msg: "{{ net.networks[0].id }}"
