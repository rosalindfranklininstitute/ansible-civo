.. Title

civo.cloud.snapshot_info module -- Gather information about Civo instance snapshots
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://github.com/rosalindfranklininstitute/ansible-civo>`_ (version 0.0.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install git+https://github.com/rosalindfranklininstitute/ansible\-civo.git`.

    To use it in a playbook, specify: :code:`civo.cloud.snapshot_info`.

.. version_added

.. rst-class:: ansible-version-added

New in civo.cloud 0.0.4

   :local:
   :depth: 1

Synopsis
--------

.. Description

- Returns details of one or all snapshots belonging to a Civo compute instance.
- When :emphasis:`name` is given, returns a single\-item list (or empty if not found).
- Uses the :literal:`civo instance snapshot` CLI commands on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: List all snapshots for an instance
      civo.cloud.snapshot_info:
        instance: web-01
        region: LON1
      register: snaps

    - name: Get a specific snapshot
      civo.cloud.snapshot_info:
        instance: web-01
        name: web-01-snap
        region: LON1
      register: snap
