.. Title

civo.cloud.size_info module -- List available Civo instance / Kubernetes / database sizes
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://github.com/rosalindfranklininstitute/ansible-civo>`_ (version 0.0.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install git+https://github.com/rosalindfranklininstitute/ansible\-civo.git`.

    To use it in a playbook, specify: :code:`civo.cloud.size_info`.

.. version_added

.. rst-class:: ansible-version-added

New in civo.cloud 0.0.1

   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Returns all sizes (plan types) available in a Civo region.
- Optionally filters by resource type using the :emphasis:`filter` parameter.
- The :literal:`name` field of each size is the value to pass to :literal:`civo\_instance`\ , :literal:`civo\_kubernetes`\ , or :literal:`civo\_database` as the :literal:`size` parameter.
- Uses the :literal:`civo` CLI binary on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: List all sizes
      civo.cloud.size_info:
        region: LON1
      register: all_sizes

    - name: List only instance sizes
      civo.cloud.size_info:
        region: LON1
        resource_type: instance
      register: instance_sizes

    - name: Show instance size names
      ansible.builtin.debug:
        msg: "{{ instance_sizes.sizes | map(attribute='name') | list }}"

    - name: List only Kubernetes node sizes
      civo.cloud.size_info:
        region: LON1
        resource_type: kubernetes
      register: k8s_sizes

    - name: List database sizes
      civo.cloud.size_info:
        region: LON1
        resource_type: database
      register: db_sizes
