.. Title

civo.cloud.database_info module -- Gather information about Civo managed databases
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://github.com/rosalindfranklininstitute/ansible-civo>`_ (version 0.0.3).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install git+https://github.com/rosalindfranklininstitute/ansible\-civo.git`.

    To use it in a playbook, specify: :code:`civo.cloud.database_info`.

.. version_added

.. rst-class:: ansible-version-added

New in civo.cloud 0.0.1

.. Deprecated


Synopsis
--------

.. Description

- Returns details of one or all Civo managed databases in a region.
- When :emphasis:`name` is given, returns a single\-item list (or empty list if not found).
- Uses the :literal:`civo` CLI binary on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Get all databases
      civo.cloud.database_info:
        region: LON1
      register: dbs

    - name: Get a specific database
      civo.cloud.database_info:
        region: LON1
        name: myapp-db
      register: db

    - name: Print database status
      ansible.builtin.debug:
        msg: "{{ db.databases[0].status }}"
