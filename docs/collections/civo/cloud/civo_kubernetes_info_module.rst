civo.cloud.civo_kubernetes_info module -- Gather information about Civo Kubernetes clusters
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `civo.cloud collection <https://galaxy.ansible.com/ui/repo/published/civo/cloud/>`_ (version 0.0.1).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install civo.cloud`.

    To use it in a playbook, specify: :code:`civo.cloud.civo_kubernetes_info`.

Synopsis
--------

.. Description

- Returns details of one or all Civo Kubernetes clusters in a region.
- When :emphasis:`name` is given, returns a single\-item list (or empty list if not found).
- When :emphasis:`kubeconfig` is :literal:`true`\ , the returned cluster dict includes the raw kubeconfig YAML.
- Uses the :literal:`civo` CLI binary on the control node.

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    - name: Get all Kubernetes clusters
      civo.cloud.civo_kubernetes_info:
        region: LON1
      register: clusters

    - name: Get a specific cluster with kubeconfig
      civo.cloud.civo_kubernetes_info:
        region: LON1
        name: my-cluster
        kubeconfig: true
      register: k8s

    - name: Save kubeconfig locally
      ansible.builtin.copy:
        content: "{{ k8s.clusters[0].kubeconfig }}"
        dest: ~/.kube/my-cluster.yaml
        mode: "0600"
